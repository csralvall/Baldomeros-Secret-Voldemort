# Baldomeros-Secret-Voldemort

# Development

## To run the Front-End server

### For the first time

```bash
sudo apt update
sudo apt install nodejs
sudo apt install npm
sudo npm install -g npm@latest
cd app/
sudo npm install
```

### From now on

```bash
sudo npm start
```
Then, open `http://localhost:3000/` on your browser

### For testing

```bash
cd app/
sudo npm run test
```
